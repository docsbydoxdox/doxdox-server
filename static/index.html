doxdox

A better way to generate documentation.

Installing doxdox

doxdox is a command line utility written in Node.js and installed via npm.

The following command will install doxdox globally on your computer allowing you to access it from any project.

        $ npm install doxdox -g

To install it on a per-project basic use the following code to install and update package.json.

        $ npm install doxdox --save

Getting Started

Basic Usage

Running doxdox without any options will walk the current directory looking for JavaScript files1 and generate documentation from the files it finds with valid JSDoc comment blocks.

doxdox will also search the current directory for a package.json file to populate both the title and description of the documentation.

This will output the contents to stdout.

        $ doxdox

To store the contents in a file use the cat shorthand operator >.

        $ doxdox > docs/index.html

Referencing a Specific File

        $ doxdox ~/git/github/Facade.js/facade.js

Referencing a Specific Directory

        $ doxdox ~/git/github/Facade.js/

Advanced Usage

Specifying a Title

        $ doxdox --title "Facade.js"

Specifying a Description

        $ doxdox --description "Drawing shapes, images and text in HTML5 canvas made easy."

Specifying an Output Path

        $ doxdox --output docs/index.html

Choosing a Layout

        $ doxdox --layout Bootstrap
        
        $ doxdox --layout Markdown

Specifying a Custom Layout

        $ doxdox --output docs/template.hbs

Specifying a Specific Package.json File

        $ doxdox --package ~/git/github/Facade.js/package.json

doxdox Online

doxdox is also available as a PaaS for single file libraries hosted on GitHub.

To use this service add the following snippet to your README.md file. Replace username and repo with the path of the project on GitHub.

The service cached documentation for 10 minutes so if your changes aren't showing up immediately wait at least 10 minutes before trying again.

        [![Latest Documentation](http://doxdox.org/images/badge.svg)](http://doxdox.org/username/repo)

Examples

  • Facade.js - Drawing shapes, images and text in HTML5 canvas made easy.
  • plastick.js - Plastick.js is a lightweight, time-accurate game loop framework for JavaScript, utilizing Facade.js.
  • CanvasToVideo - An experiment in converting Canvas animations to video.

  1. ↩ doxdox will ignore the following directories: .git, node_modules, bower_components, and test. It will also ignore files matching .min, gruntfile, and gulpfile.
